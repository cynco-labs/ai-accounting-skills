# financial-statements-accounting

Prepare primary statements, notes to the financial statements, compilation reports, and Excel working-paper workbooks.

## Skills

| Skill | Purpose | Command |
|---|---|---|
| `prepare-primary-statements` | SOFP/SOCI/SOCE/SCF | `/financial-statements-accounting:prepare-primary-statements` |
| `prepare-notes` | Notes to FS | `/financial-statements-accounting:prepare-notes` |
| `compilation-report` | ISRS 4410 | `/financial-statements-accounting:compilation-report` |
| `generate-workbook` | Excel workpapers | `/financial-statements-accounting:generate-workbook` |
