# /compilation-report

## Purpose

ISRS 4410 compilation report — **no assurance**.

Load `references/compilation_report.md`.

## Contents
- Addressee (management / directors)
- Management responsibilities
- Accountant responsibilities
- Statement that no audit or review was performed; no assurance expressed
- Framework reference (MPERS etc.)
- Signature block (firm name, date)

## Do not
- Imply audit/review
- Omit scope limitations that were agreed
