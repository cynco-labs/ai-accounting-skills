# finalisation-accounting

Lock accounts, management approval packs, auditor deliverables, and statutory/tax filing handoff from finalised figures.

## Skills

| Skill | Purpose | Command |
|---|---|---|
| `finalise-accounts` | Lock books | `/finalisation-accounting:finalise-accounts` |
| `management-approval` | Directors approval | `/finalisation-accounting:management-approval` |
| `auditor-pack` | Audit handoff | `/finalisation-accounting:auditor-pack` |
| `statutory-handoff` | Tax/SSM handoff | `/finalisation-accounting:statutory-handoff` |

## Setup
