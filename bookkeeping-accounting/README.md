# bookkeeping-accounting

Record sales, purchases, receipts, payments and journals; classify every transaction against the chart of accounts; produce balancing journal entries.

## Skills

| Skill | Purpose | Command |
|---|---|---|
| `record-transactions` | Extract & record | `/bookkeeping-accounting:record-transactions` |
| `classify-transactions` | COA classification | `/bookkeeping-accounting:classify-transactions` |
| `chart-of-accounts` | COA setup | `/bookkeeping-accounting:chart-of-accounts` |
| `journal-entries` | Double-entry journals | `/bookkeeping-accounting:journal-entries` |
