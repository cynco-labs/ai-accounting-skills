# mpers-accounting

MPERS (and MFRS) technical review of significant balances and a disclosure completeness checklist before financial statements are finalised.

## Skills

| Skill | Purpose | Command |
|---|---|---|
| `mpers-technical-review` | Standards review | `/mpers-accounting:mpers-technical-review` |
| `disclosure-checklist` | Notes completeness | `/mpers-accounting:disclosure-checklist` |

## Setup

Firm-wide: `/engagement-accounting:cold-start-interview`

Runtime config: `~/.claude/plugins/config/claude-for-accounting/mpers-accounting/CLAUDE.md`
