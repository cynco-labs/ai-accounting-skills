Inside standards-aware classify

Credits on the bank are not auto-“Sales”. Outflows are not auto-“expense”.

💰

Money in → revenue?

Walk MPERS S23 / MFRS 15: goods vs services, principal vs agent, deposits vs earned. Write revenue_recognition.md.

🏗️

Money out → asset?

Capitalise vs expense, repairs, software, owner personal. Write capital_vs_expense.md (FAR hooks if PPE).

❓

Only hard questions

≤3 structured questions per theme — not 40 line-by-line chats. Answers saved; codes follow the conclusion.
