# tax-accounting

Malaysian tax computations (Form B/C/P/PT/TF/TP), capital allowances, and statutory deduction checks (EPF/SOCSO/EIS/PCB).

## Skills

| Skill | Purpose | Command |
|---|---|---|
| `tax-computation` | ITA computation | `/tax-accounting:tax-computation` |
| `capital-allowances` | CA schedule | `/tax-accounting:capital-allowances` |

## Setup

Firm-wide: `/engagement-accounting:cold-start-interview`

Runtime config: `~/.claude/plugins/config/cynco-accounting-skills/tax-accounting/CLAUDE.md`
